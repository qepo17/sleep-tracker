* Initiate development environment

- Install devbox

https://www.jetify.com/docs/devbox/installing_devbox/

- Setup ENV

See what's inside `devbox.json` env section

- Run devbox

```bash
devbox shell
```

- Run migrations

```bash
rails db:migrate
```

* Database initialization

- Init DB

```bash
devbox run init_db
```

- Run DB

```bash
devbox run start_db
```

* How to run the test suite

```bash
devbox run run_test
```
